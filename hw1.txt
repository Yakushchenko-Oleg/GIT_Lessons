1. Как скопировать на свой компьютер репозиторий из github.com?
 - Для того чтоб скопировать репозитприй к себе в рабочий каталог, нужно ввести команду git clone и указать ссылку к репозиторию.

2. Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?
 - нужно: 
	перейти в теритнале в необходимую дирикторию
	ввести команду git init - инициализировать (подключить) директорию для работы с git сервером
	ввести команду git add b и указать имя (имена) файла или поставить "." чтоб выбрать все файлы в директории и добавить их в "индекс"
	Ввести команду git commit с ключем -м для добавления комментария вковычках после ключа. Эта команда добавляет слепок добавленных в индекс файлов в коммит и делает этот коммит родительским для следующих.
	ввести команду git push чтоб передать данные переданные в последний коммит из локального репозитория в git репозиторий.

3. Как получить на локальный компьютер изменения с github.com?
 	перейти в теритнале в ранее проинициализированую дирикторию
	ввести команду git pull она она добавит изменения из git репозитория привязанного к дирректории в локальный репозиторий в виде коммита.

4.  - команда git status отображает состояние рабочего каталога и раздела проиндексированных файлов. С ее помощью можно проверить какие файлы были добавлены в индексацию

 - команда git log позволяет просмотреть историю загрузок коммитов в git репозитарий и обратно

 - команда git remote -v показывает с каким репозитарием в git связан локальный репозитарий

5. Как вернуть файл из индекса в рабочий каталог?
 - ввести команду git restore --staged

6. Как отменить ошибочный commit ?
- команда git reset HEAD~1 перемещает указатель текущей ветки на указанное количествр коммитов назад тем самым по сути удаляет коммиты из истории

7. Как исправить коммит отправленный на сервер GitHub?
	Вводим  команду git revert head (добавляем новый комит с действиями обратными посленнему (ошибочному) комиту.
